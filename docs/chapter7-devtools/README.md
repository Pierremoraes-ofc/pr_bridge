# Ferramentas de Desenvolvedor (DevTools)

O `pr_bridge` não é focado só no jogador final. Ele brilha nas ferramentas feitas para *quem cria os scripts*.
Esqueça a necessidade de abrir softwares de terceiros ou mapas complexos para criar sistemas. Você pode fazer tudo dentro do jogo com as APIs nativas do Bridge!

Com os módulos em `pr_lib.fivem.devtools` e `devlaser`, você consegue extrair coordenadas perfeitamente alinhadas, testar objetos e ver debugs na tela.
